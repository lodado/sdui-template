# React Grid Layout SDUI Example

This Next.js app demonstrates a server-driven UI (SDUI) grid layout using `react-grid-layout` and the SDUI component library. It showcases:

- Grid layout state stored in the SDUI document.
- Drag/resizing updates back to SDUI state.
- Shape tiles that toggle between square, triangle, and circle.

## Scripts

```bash
pnpm --filter react-grid-layout-example dev:internal
pnpm --filter react-grid-layout-example build:internal
```
