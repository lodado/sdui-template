# @lodado/sdui-document-react

Hybrid Notion-like editor bindings for `@lodado/sdui-document`: focused-block ProseMirror inline editing plus React block renderers.

## Installation

```bash
pnpm add @lodado/sdui-document-react @lodado/sdui-document
# peer: react, react-dom
```

## Usage

```tsx
import { SduiDocumentEditor } from '@lodado/sdui-document-react'

// See Storybook DocumentEditor stories for full setup examples.
```
